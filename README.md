# 🌍 LAD-AI

## Language Acquisition Device

LAD-AI is an AI-powered language translation application developed using Python and Gradio.

## Features

- English → French Translation
- English → Yoruba Translation
- English → Igbo Translation
- English → Hausa Translation
- Translation History
- Favorites
- User Login System
- SQLite Database
- Dictionary
- Statistics Dashboard

## Technologies Used

- Python
- Gradio
- SQLite
- Deep Translator
